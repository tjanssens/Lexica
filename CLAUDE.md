# Lexica - Development Notes

## Opstarten

### Backend (API)
```bash
cd src/Lexica.Api
dotnet run --launch-profile https
```
- HTTPS: https://localhost:7105
- HTTP: http://localhost:5066

### Frontend (Angular)
```bash
cd src/lexica-frontend
npm start
```
- URL: http://localhost:4303
- Frontend verwacht de API op https://localhost:7105
